# ✈️ Airline Executive Insights Dashboard

A complete end-to-end Data Analytics project that transforms raw airline data into actionable business insights using Python and Streamlit.

---

## 📊 Project Overview

This project focuses on analyzing airline ticket data to uncover pricing trends, route performance, and customer patterns.  
It demonstrates the full data workflow — from data cleaning to interactive dashboard development.

---

## 🚀 Features

- 📌 Interactive dashboard with real-time filters  
- 💰 Key metrics: Total Flights, Revenue, Avg/Min/Max Price  
- 🛫 Route-level insights (Source → Destination)  
- 📉 Price distribution analysis  
- 🧭 Airline comparison & travel class insights  
- 🎯 Clean and modern UI design  

---

## 🛠️ Tech Stack

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Streamlit**
- **Jupyter Notebook**

---

## 🧹 Data Cleaning (Jupyter Notebook)

- Handled missing values  
- Standardized column names  
- Converted categorical data (e.g., stops → numeric)  
- Prepared dataset for analysis
